This directory contains avatar images for blog post authors. Each author referenced in blog posts should have an avatar image.

## Image Specifications

**Dimensions**: 200x200px (square)
**Format**: JPG or PNG
**File Size**: Under 50KB
**Content**: Professional headshot or avatar

## Placeholder Images

Until custom images are created, you can use:
- Professional avatar generators (UI Avatars, Boring Avatars)
- Stock professional headshots
- Generic professional avatars
- Initials-based avatars

## Avatar Generators

Free avatar generation tools:
- UI Avatars: https://ui-avatars.com/
- Boring Avatars: https://boringavatars.com/
- DiceBear: https://www.dicebear.com/

## Adding Author Images

1. Create or download professional avatar image
2. Resize to 200x200px square
3. Optimize for web (compress to under 50KB)
4. Rename to match the filename referenced in blog post frontmatter
5. Place in this directory (`public/images/authors/`)
6. Verify image displays correctly on blog posts

## Example: Creating Placeholder with UI Avatars

For temporary placeholders, you can use UI Avatars:
```
https://ui-avatars.com/api/?name=Emily+Rodriguez&size=200&background=0D8ABC&color=fff
```

Save the generated image with the appropriate filename.
