# Blog Featured Images

This directory contains featured images for blog posts. Each blog post requires a featured image referenced in its frontmatter.

## Image Specifications

**Dimensions**: 1200x630px (optimal for social sharing)
**Format**: JPG or PNG
**File Size**: Under 200KB (optimized for web)
**Content**: Healthcare-related imagery that represents the blog post topic

## Placeholder Images

Until custom images are created, you can use:
- Stock photos from healthcare-focused sites (Unsplash, Pexels)
- Generic healthcare imagery (stethoscope, clinic, medical professionals)
- Abstract healthcare-themed graphics

## Image Sources

Recommended free stock photo sources:
- Unsplash (https://unsplash.com/s/photos/healthcare)
- Pexels (https://www.pexels.com/search/healthcare/)
- Pixabay (https://pixabay.com/images/search/healthcare/)

## Adding Images

1. Download or create image
2. Optimize for web (compress to under 200KB)
3. Rename to match the filename referenced in blog post frontmatter
4. Place in this directory (`public/images/blog/`)
5. Verify image displays correctly on blog listing and detail pages
